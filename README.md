InfoPelis

InfoPelis es una página web en construcción dedicada a brindar información sobre películas populares, incluyendo algunos datos básicos.
El objetivo es ofrecer a los visitantes una experiencia clara y atractiva para descubrir y explorar películas de distintos géneros.

Características

Página de inicio con películas destacadas y con su página dinamica. Sección “Nosotros” donde se explica la misión del proyecto y su estado de desarrollo.
Blog con noticias, curiosidades y enlaces externos relacionados con películas. Página de contacto para que los visitantes puedan comunicarse o sugerir contenido. 
Diseño responsive, adaptado a desktop, tablet y celular. Uso de Tailwind CSS para estilos modernos y consistentes. Menú hamburguesa en móviles para una navegación más cómoda.

Tecnologías Usadas

Astro para la estructura de la página y Tailwind CSS para los estilos y diseño responsive.

Estado del Proyecto

Actualmente, InfoPelis está en desarrollo. Algunas secciones tienen contenido de prueba, y se planea agregar más películas, noticias y enlaces a fuentes oficiales.
